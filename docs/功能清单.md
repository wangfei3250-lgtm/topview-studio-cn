# 功能清单

## 首页

- 左侧导航：首页、Agent、画布、短剧工作室、工作台、图片、视频、数字人、音频、技能
- 顶部智能体选择：Canvas、V2、V1
- 首页快捷入口：创作短剧、营销视频、图片设计、视频复刻
- 首页模板卡：UGC 广告、批量带货广告、视频复刻、商品链接视频、TVC、虚拟试穿、餐厅介绍、迷你故事
- 底部分类入口：Social Content、Ad Video、Ecommerce、Film、AI UGC Video、Explainer Video、Corporate Video、Local Services、Local Picks、TikTok Ad Library、All Tools

## 智能体

- Canvas：全流程视频创作
- V2：快速短片规划
- V1：轻量视频草案
- 输入消息后可生成智能体计划并写入工作台任务
- 上传素材按钮会把素材加入侧栏参考列表

## 技能库

- 两列技能卡列表
- 分类筛选：全部、剧情、广告、电商、社媒、UGC、本地服务、TikTok、图片
- 每张技能卡支持“直接使用”和“自动”开关
- 直接使用会进入 Agent，并把该技能转换成执行计划

## 短剧工作室

- 文本或文件输入
- 单集/连续剧切换
- 编剧等级选择
- 生成后进入制作流程
- 制作流程包含角色与场景、剧情节奏、故事板、配音字幕、最终视频

## 画布

- 画布列表
- 新建画布
- 画布编辑器弹层
- 分镜帧操作：重绘、修补、重光、放大
- 导出画布会写入工作台任务

## 工具模块

- 图片工具：文生图、图片编辑、故事板、局部重绘、角色替换、换脸、超分、角度编辑
- 视频工具：图生视频、文生视频、全参考、角色替换、超分、运动控制、修复、去水印字幕
- 数字人工具：AI Avatar、Product Avatar、Design My Avatar、Video Lip Sync、Virtual Try-On、Face Swap
- 音频工具：Voiceover、AI Music、Text to Speech、Instant Voice Clone、Voice Design
- 工具卡片会打开任务弹层并写入工作台任务

## 工作台

- 任务状态统计：生成中、排队中、已完成、剩余积分
- 任务会模拟从排队中变为生成中，再变为已完成
